README

Ce projet est un jeu 2D simple qui utilise la minilibx linux, des textures, des sprites et quelques éléments basiques de gameplay.

Toutes les maps sont dans le dossier maps.


	make

	make bonus

	make clean

	make fclean

	./so_long maps/map1.ber
